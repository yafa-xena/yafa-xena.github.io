2020年终总结

2021-02-10
关于过去的2020年 今天是除夕，正好今天过年的准备也做完了，有时间好好去回顾过去的一年。 过去的2020年是魔幻的开端，因为疫情的缘故我能够出去找工作的时间就很靠后。 虽然是找了一家公司，但是这家公司也让我明白了小说里面的场景有些地方是真实的，就是魔幻现实的映射。 在工作上获得的进步不是很大，刚开始接触到这个运维行业很多地方都是在探索和学习，虽然感觉进度有点跟不上自己的计划，但是获奖也是一个小惊喜啦。 在生活上也是上当受骗了一把，相信所谓的消费主义深受其害。每个月都是月光的状态，对于未来的风险承受能力很低，只能够祈祷不能把当前的工作给丢掉，留给自己审视的时间是很少的。 在经历这么几个月的“上流月”最后醒悟了（感谢学姐！），不能这么继续下去了。 在Youtube上看到了一些关于极简主义的视频之后就爱上了，开始做一些减法。像是原来的衣柜都是满的，都是当时一些换季打折啦，或者是促销的时候买的。现在与其说买那么多早上不知道穿什么，还不如说就保留自己最为中意的几件衣服，针对于不同的场景。保留了3个场景的衣服分别是：居家、外出运动、上班。其他的衣服统统挂闲鱼卖掉了，这就是另外一件事情了。还有一些过剩的化妆品（看小红书看上头买的），一些多余的电子产品，这些都是削减我荷包的主要物品，在内心取恒之后最终将这些中非必要的（非日常需要，心动值不高）都挂闲鱼卖掉了。 在做完了对生活的减法之后就有了空间开始思考自己的职业发展和人生发展，虽说做这个运维行业只是一时兴起，但是现在做下来之后发现是一件很有意思的事情，很喜欢在这个像是F1方程式赛车一样的过程，你需要考虑到用哪些开源的软件作为一些基础层，哪一些适合作为中间层，哪些是适合做管理层。将其组合在一块之后这就是公司的F1方程式（产品）。这辆方程式将会在赛道上和其他公司的方程式进行比赛，都是想要争取那个第一名，从而获得市场和相关的声誉非物质资产。通过这个视角再去看公司里面的其他人的角色在这个赛道上是做什么的？以及现在他们所做的是否符合自己的角色设定，这个非常有趣在这个里面可以找到破局而出的关键要素。 现在看来我现在所处的公司有很多地方是不合理，像是没有意义的996，不论是推式沟通还是拉式沟通还是面对面地沟通几乎没有，团队气氛上不是那么的积极向上。组织形态上也从最初的学习型组织向着一种淤泥化转变。这也给我做出警醒应该做出选择是时候做出改变环境或者是改变自己（在当前环境下提出变革或者是换一家公司同时也应该做更强大的自己）。 听学姐说过关于运维的发展方向: DevOps DevOps是一种运营哲学，旨在提高构建和交付效率上。DevOps实践可以促进团队之间的集成，开发和运维之间的沟通成本降低。这个领域的专家会寻找效率低重复的流程，并用脚本或者是一些自动化工具去替换。 SRE SRE工程师最重视的是业务运行的时间和正确性。监控网络，部署软件，Oncall，计划未来的扩展和调试中断等等。这个需要会一定的代码并且还需要有坚实的网络和服务理解。 Security operations engineers 不知道能不能缩写成SOE，安全运维注重于信息安全的日常工作。像是安装和操作一些可以搜索漏洞和探测网络攻击的工具。还会参与攻击模拟从而来评估当前的防御技术的有效性。 Network Administrator 也就是所谓的网管了，主要负责设计网络，配置和运行。但是现在公司很少去招这方面的人一般都是网络提供商给解决方案，主要招的还是在数据中心，因为数据中心需要管理各种网络设备：交换机、路由器、防火墙等等。 Database administrators 这个是我们常常听到的DBA，主要负责是安装和管理数据库。管理数据库的架构，配置集群，调整设置以获得最佳性能，并且帮助开发制定有效的查询。DBA通常要有关系型和非关系型数据库的经验。 Network operations center （NOC） engineers NOC工程师主要是在数据中心检测大型站点的运行情况，进行事件跟踪。最为常见的就是要看显示图形和指标的那面监视墙。 Data center technicians 数据中心技术员 这个角色主要负责的是跟踪设备的生命周期，在机房里面部署服务器，以及处理日常的工单。 Architects 架构师 架构师意味着在多个领域有了足够深厚的专业知识。他们会利用自己的经验来设计系统。他们的工作可能会包含如何定义安全区域和分段，消除单点故障，规划未来的增长，确保多个网络和相关方之间的连通性以及其他范围的决策。 从自身的积累和水平来看我认为之后的发展可以先从SRE做起，再去补充网络的知识，以架构师为导向。 方向选择已经做出来了，那么下一步就是很明确了，如何成为那个自己。 2021年 做更强大的自己 过去未去，未来以来。过去一年中的疫情影响还是陪伴着我们。我开始思考新的一年如何开始，如何靠近我的目标。我找到的答案是做更强大的自己。 说起来是非常空洞的，我们如何去做更强大的自己呢？我们每个人的心中都潜藏着属于自己的梦想，而那个梦想里面的自己就是那个更强的自己，我想要叫这个自己为超级自我。 尝试去代入这个超级自我，我会发现一些习惯（行为模式）上的不同，我发现正是因为习惯上的不同和思维模式上的区别造就了当前自我和那个超级自我之前的距离。接下来的一年里面我也会以这个为切入点尝试对自己的行为习惯上做一些改变。 对于已经开始的2021年我对于自己有这些期望： 学习 事业 生活 学习 在学习这个方向上我希望我能够培养出好的3个习惯： 每个月挑战看2本书 每个月更新2篇blog 每天的日记 第一个习惯每个月看两本书对于我来说不是特别难以完成的任务，但是也不是太过于简单的任务。刚刚好有一种挑战需要努努力才可以达到，不会说太过于简单不想要去行动或者是太难没有办法完成目标的那种失落。 第二个习惯是关于写作的，在这个写作的过程中发现这是一种思考的过程，在写作的时候之前学过的东西有一些疑问的点就自然出来了，因为有了一种换位的感觉，如果我的读者或者老师朋友看到这篇文章是否能够体会到我的想法，能够将我想要传播的点接收并且由此展开一些更多的观点和思考。总而言之我很喜欢写作，不论是过程还是最终的结果都让我很开心，我想要继续保持这个写作的习惯。 第三个习惯是关于每天写日记，最近学姐给我推荐了一本《晨间笔记的奇迹》这本书中提到的一个观点就是在早上记笔记。通过几天的尝试感觉确实有点不一样，对于写日记没有了抗拒的感觉，早上起来跑完步的时候自然而然或者是在跑步的途中就开始思考昨天的一些事情以及今天要做那些事情（随身带一个小本子随时收集）将这些记录下来，就像是在一堆服务器中找到了你按了提示灯的那台一样。我希望我可以保持这个习惯。 在学习这个方向上我希望我能够逐渐消除的一些习惯： 拖延 我在学习上有很严重的拖延症，假设我想要完成一门课程，距离测试可能可能只剩下2天了我是那种会拖延到截止提交的最后2小时，然后满分通过的（不用想了肯定是很差）。 当像是这样的事情次数多了之后开始有一种对学习的厌倦感，我非常想要改变这种行为模式。现在开始做一些尝试，类似于找到一个合适的时间段，排除掉所有可能干扰到学习的因素，像是手机电脑之类的，然后专心学习。
